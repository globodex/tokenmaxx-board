Profile management

First-time ambassadors use `/update-stats setup` to create a row with optional location metadata. Returning ambassadors use `/update-stats` to refresh only the five visible stats. Codex syncs the shared JSON file in the Globodex repo through GitHub.

npm run sync:profile -- --name "Ada Launch" --handle "@ada.launch" --location "San Francisco" --country "United States" --lifetime 12.4B --peak 1.3B --task "17h 42m" --current 42 --longest 58
Tokenmaxx Board is unofficial. Stats are opt-in and self-reported from visible Codex profile pages.
